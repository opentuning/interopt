====================================
Installation
====================================

Requirements
-----------
- Python 3.8+
- gRPC
- pandas
- numpy
- scikit-learn
- FastAPI (for API server)

Installation Steps
----------------

1. Install using pip::

    pip install interopt

2. Or install from source::

    git clone https://github.com/your-repo/interopt.git
    cd interopt
    pip install -e .
