InterOpt: An Interoperability Layer for Black-box Optimization

InterOpt provides a unified interface for running black-box optimization experiments across different hardware and software backends. It supports both synchronous and asynchronous execution, tabular datasets, and surrogate models.

Features

  • Unified interface for optimization experiments

  • Tabular dataset management

  • Surrogate model integration

  • Asynchronous execution support

  • GRPC-based communication
